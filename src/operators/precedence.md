# Operator Precedence

| Precedence | Operators |
| --- | --- |
| 1 | Assignment (`+=`, `-=`, ...)
| 2 | Post-fix (`.x`, `()`, `?`, `[]`, `->x()`) |
| 3 | Pre-fix (`-`, `!`, `~~`, `~`, `*`, `&`) |
| 4 | `??` |
| 5 | `%` |
| 6 | `*`, `/` |
| 7 | `+`, `-` |
| 8 | `&`, \|, `^`, `<<`, `>>`, `>>>` |
| 9 | `<=`, `<`, `>=`, `>` |
| 10 | `==`, `!=` |
| 11 | `&&`, \|\| |
| 12 | \|>, `..` |
